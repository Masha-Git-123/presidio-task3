# React Dashboard Application

## Overview

This project is a React-based Dashboard Application developed using modern frontend technologies. It demonstrates authentication, protected routing, form handling, API integration, data visualization, and state management.

## Features

* User Login Page
* Protected Routes
* Role-Based Access Control
* Dashboard Page
* React Query for API Data Fetching
* Chart.js Bar Chart
* Chart.js Line Chart
* React Hook Form
* Zod Form Validation
* Zustand State Management
* TypeScript Support
* Responsive UI

## Technologies Used

* React
* TypeScript
* Vite
* React Router DOM
* TanStack React Query
* Chart.js
* React ChartJS 2
* React Hook Form
* Zod
* Zustand

## Installation

Clone the repository:

git clone <https://github.com/Masha-Git-123/presidio-task3.git>

Install dependencies:

npm install

Run the project:

npm run dev

Build for production:

npm run build
